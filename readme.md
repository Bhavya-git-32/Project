# Automated Bulk WhatsApp Messaging Tool

A Python-based automation tool to send WhatsApp messages to multiple recipients simultaneously, overcoming the default WhatsApp limit of 5 messages at a time.

## Features
- Send messages to **up to 150 recipients** simultaneously.
- Supports scheduling of messages for efficient communication.
- Handles errors and exceptions to ensure **reliable message delivery**.
- Modular and reusable code for future enhancements.

## Technologies Used
- Python
- WhatsApp Web Automation (selenium or pywhatkit)
- Threading/Concurrency for handling multiple recipients
